使用帮助

时间序列初始化处理

  1. 上传CSV文件：将包含时间序列数据的CSV文件上传到系统。
  2. 处理CSV数据：系统将自动分析文件并生成敏感版和不敏感版的分析结果。
  3. 查看分析结果：在结果区域可以查看分析生成的图表和数据文件。
  4. 编辑CSV数据：如需修改数据，可以使用编辑功能调整CSV内容。

趋势复盘分析

  1. 上传已处理的分析数据：将已完成初始化处理的CSV文件上传。
  2. 设置分析查询：输入您希望进行的分析查询内容。
  3. 开始复盘分析：系统将根据查询内容进行深度分析并生成报告。
  4. 查看分析日志：在日志区域可以实时查看分析进度和结果。
